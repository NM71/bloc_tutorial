# bloc_tutorial

BLOC Tutorial

## Concepts Used

- CUBIT: Simplified version of BLOC for easy state management.
- BLoCProvider: Provides CUBIT to VIEW (UI)
- BLoCBuilder: UI
- BLoCListener: Reacts to State Changes
